# Project Summary
AmanahLife is a comprehensive Islamic lifestyle management SaaS platform designed to empower Muslim individuals and families in managing their spiritual, financial, and wellness needs. Key features include prayer tracking, financial management, and wellness monitoring, all aimed at enriching daily lives and spiritual practices. The platform is built on a Node.js + Supabase architecture, ensuring scalability and maintainability for rapid feature development and an enhanced user experience.

# Project Module Description
The project consists of several functional modules:
- **Dashboard:** Displays daily summaries including tasks, prayers, goals, and balance.
- **Tasks:** Manages tasks with recurrence, priorities, and family sharing.
- **Goals:** Tracks personal, financial, spiritual, and family goals with progress monitoring.
- **Finance:** Handles income/expenses, budgets, and includes a Zakat calculator.
- **Prayer:** Provides prayer times, tracking, and consistency charts.
- **Spiritual:** Features Quran tracking, Dhikr counter, Duas collection, and Ramadan tools.
- **Wellness:** Monitors mood, sleep, hydration, and stress levels with daily logging.
- **AI Insights:** Offers personalized tips and monthly life reviews based on user data.
- **Planner:** Provides agenda, weekly, and monthly views for task management.
- **Settings:** Manages user preferences including theme, language, and notifications.
- **Notifications:** Customizable alerts for tasks, prayers, and goals.
- **Subscriptions:** Manages user subscriptions through Stripe.
- **Data Export:** Allows exporting financial and goal data as CSV files.
- **RTL Support:** Supports right-to-left language layouts.

# Technology Stack
- **Frontend:** React 18, Vite, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend:** Node.js, Supabase
- **Database:** PostgreSQL (Supabase)
- **Payment Processing:** Stripe
