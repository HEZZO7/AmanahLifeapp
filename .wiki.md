# Project Summary
AmanahLife is a comprehensive Islamic lifestyle management SaaS platform designed to help Muslim individuals and families manage their spiritual, financial, and wellness needs. The platform integrates features such as prayer tracking, financial management, and wellness monitoring, aiming to enhance users' daily lives and spiritual practices. With a transition to a Node.js + Supabase architecture, the project focuses on scalability and maintainability, facilitating rapid feature development for an improved user experience.

# Project Module Description
The project consists of several functional modules:
- **Dashboard:** Displays daily summaries including tasks, prayers, goals, and balance.
- **Tasks:** Manages tasks with features for recurrence, priorities, and family sharing.
- **Goals:** Tracks personal, financial, spiritual, and family goals with progress monitoring.
- **Finance:** Manages income/expenses, budgets, and includes a Zakat calculator.
- **Prayer:** Provides prayer times, tracking, and consistency charts.
- **Spiritual:** Features include Quran tracking, Dhikr counter, Duas collection, and Ramadan tools.
- **Wellness:** Monitors mood, sleep, hydration, and stress levels with daily logging.
- **AI Insights:** Offers personalized tips and monthly life reviews based on user data.
- **Planner:** Provides agenda, weekly, and monthly views for task management.
- **Settings:** Manages user preferences including theme, language, and notifications.
- **Notifications:** Customizable alerts for tasks, prayers, and goals.
- **Subscriptions:** Manages user subscriptions through Stripe.
