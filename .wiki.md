# Project Summary
AmanahLife is a comprehensive SaaS platform designed to support users in managing their spiritual, financial, and wellness needs. It offers integrated features such as prayer tracking, financial management, and wellness monitoring, enhancing users' daily lives and spiritual practices. The project is currently transitioning to a Node.js + Supabase architecture to improve scalability, maintainability, and accelerate feature development, thereby providing a more robust user experience.

# Project Module Description
The project consists of several functional modules:
- **Dashboard:** Displays a life score based on prayer, tasks, health, and finance.
- **Planner:** Manages tasks with recurrence, priorities, and family sharing.
- **Finance:** Tracks income/expenses, budgets, and includes a Zakat calculator.
- **Spiritual:** Offers prayer tracking and Quran page management.
- **Ramadan Mode:** Provides tools for fasting and Eid budgeting.
- **Family:** Facilitates group management of tasks and budgets.
- **Wellness:** Monitors mood, sleep, hydration, and stress levels.
- **AI Assistant:** Utilizes user data for personalized insights.
- **Life Reviews:** Generates AI-driven reports for premium users.
- **Subscriptions:** Manages user subscriptions through Stripe.
- **Authentication:** Supports email/password and Google OAuth login.
- **New Features:** Includes Prayer Times, Quran Reader, Dhikr Counter, and Duas Collection.

# File Description Inventory
- **AmanahLife_App_Description.md:** Overview of the application, its features, and user benefits.
- **AmanahLife_Atoms_Prompt_v2.md:** Technical instructions and prompts for developers.
- **IMG-20260519-WA0XXX.jpg:** Screenshots showcasing the user interface and design aspects.
- **package.json:** Lists project dependencies and scripts for building and running the application.
- **App.tsx:** Main application component that sets up routing and context providers.
- **AuthCallback.tsx:** Handles the authentication callback from Supabase.
- **Index.tsx:** Home page displaying user actions and navigation.
- **Login.tsx:** User login and signup interface.
- **PrayerTimes.tsx:** Displays prayer times based on user's location.
- **QuranReader.tsx:** Allows users to read and track progress in the Quran.
- **DhikrCounter.tsx:** Digital counter for dhikr with progress tracking.
- **DuasCollection.tsx:** Collection of categorized duas with bookmarking.
- **AuthContext.tsx:** Manages authentication state and methods.

# Technology Stack
- **Frontend:** React 18, Vite, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend:** Node.js, Supabase
- **Database:** PostgreSQL (Supabase)
- **Payment Processing:** Stripe
